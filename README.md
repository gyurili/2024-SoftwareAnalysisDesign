# 교과목 개요
</br> 
소프트웨어 개발시 가장 중요한 과정 중 하나가 요구 사항을 분석하고 프로그램의 기본 구조를 설계하는 작업이다. 본 강의에서는 소프트웨어 개발시 자주 발생하는 여러 가지 설계 문제에 대한 해결책을 제공하는 '디자인패턴'을 자바 언어를 중심으로 학습한다. 또한 다양한 실습 과제를 통해서 실제 프로그래밍 과정에 디자인 패턴을 적용하는 방법을 익힌다.
